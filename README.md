## 测试系统
在Win10-1909 Win10-1903,只有狠人才会在Win7上进行测试

## 工作原理
1.修改驱动程序的派遣函数(兼容性强)

2.定位到物理内存直接修改硬件数据(兼容性弱)

## 关于蓝屏
1.看懂内核代码

2.用WinDbg定位到蓝屏代码后进行修复

3.什么? 不会? 那你还要好好学哈
